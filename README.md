# sasohan-match

Matching API server of [sasohan](https://github.com/insanesclub/sasohan)

## [API specification](https://github.com/insanesclub/sasohan/wiki/match)

## Supported functions

- [x] connect
- [x] disconnect
- [x] match

**Install**

```bash
$ GO111MODULE=on go get github.com/insanesclub/sasohan-match
```

**Build**

```bash
$ make
```
